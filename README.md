# Dragonpedia

Dragonpedia is a game where you need to identify as many dragons as possible!!

Check out the demo here: https://kittycat81.itch.io/dragonpedia

Controls:

Jump - Space/Up arrow/W/Xbox A
Move Right - Right arrow/D/Xbox left joystick right
Move Left - Left arrow/A/Xbox left joystick left
Shoot Fire - Q/Xbox left trigger
Shoot Ice - E/Xbox right trigger
Move mouse (on Xbox) - move right joystick
Click/Accept (on Xbox) - A button
Restart level - Y/Xbox Y
